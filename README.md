# SmartStack Search

SmartStack Search is a simple, responsive web project that integrates **Google Custom Search Engine (CSE)** with **Google AdSense** for monetization. It provides a modern UI built using **Bootstrap 5**, allowing users to perform personalized searches powered by Google.
